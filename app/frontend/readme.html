DuraCloud Content Retrieval Tool

A modern web interface for downloading content from DuraCloud storage spaces.

Features

Modern Web Interface

Real-time progress tracking and intuitive controls

Batch Downloads

Support for content list files and bulk operations

Secure Handling

Secure credential handling for DuraCloud access

Quick Start

Prerequisites

  • Docker Desktop
  • PowerShell (Windows) or Terminal (macOS/Linux)

Installation Steps

1. Clone the repository:

git clone https://github.com/kgitbit/duracloud-retrieval-web.git
cd duracloud-retrieval-web

2. Configure environment (optional):

cp app/.env.template app/.env
# Edit app/.env with your preferred settings

3. Start the application:

# Windows PowerShell
.\docker-commands.ps1 rebuild

# macOS/Linux
docker compose -f app/docker/docker-compose.yml up --build -d

Project Structure 

duracloud-retrieval-web/
├── app/
│   ├── docker/             # Docker configuration
│   │   ├── Dockerfile     # Container definition
│   │   └── docker-compose.yml
│   ├── frontend/          # Web interface
│   │   └── index.html    # Main interface
│   ├── server.py          # Flask server
│   └── requirements.txt   # Python dependencies
├── downloads/            # Downloaded files
├── duracloud-logs/       # Application logs
└── content/             # Content list files

Usage

  1. Open http://localhost:5000 in your browser
  2. Enter your DuraCloud credentials
  3. Select your space and upload a content list file
  4. Choose download directory
  5. Start the download process
  6. Monitor progress in real-time

Docker Commands

Windows/Linux/macOS
cd app/docker
docker compose up --build -d    # Build and start containers
docker compose down            # Stop containers
docker compose logs           # View logs
docker compose ps             # Check status

Troubleshooting

Error: Port 5000 already in use

Solution: Change port in docker-compose.yml or stop conflicting service

Error: Permission denied for downloads directory

Solution: Check directory permissions and ownership

Error: Unable to connect to DuraCloud

Solution: Verify credentials and network connectivity
